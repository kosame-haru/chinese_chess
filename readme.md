# 中国象棋
这里是由kosame haru（负责全部程序和音乐）、久弥紀伊（负责美工制作）制作的中国象棋小游戏，预定实现基本走法、人机对战、残棋加载等功能。希望大家喜欢。 

***

如果想要了解如何编译，请参考 `manual\compile.md`. 

如果想要了解命令行 (CLI) 版本如何操作，请参考 `manual\cli.md`. 

如果想要了解图形化 (GUI) 版本如何操作，请参考 `manual\gui.md`. 

如果想要了解象棋棋谱格式、棋局格式，请参考 `manual\format.md`.
